# 设计规范

本项目的 UI/交互设计需遵循团队统一的设计规范体系。

## 设计规范来源

```
（待填写：如 git@xxx.git 或 Figma 设计系统链接）
```

## 使用方式

在 AI 会话中说"加载设计规范"或"获取设计 token"，AI 会：

1. 读取上述仓库/链接内容
2. 列出可用的设计规范文件（如色彩、字体、间距、组件等）
3. 由你选择需要引入到当前项目的规范
4. 将选中的规范文件拉取到本目录下

## 规范内容（待加载）

执行上述流程后，选中的设计规范文件会出现在本目录中。

## Design Token 对接

当项目升级为完整 Agentic 底座（通过 `bootstrap-agentic-project`）后，Design Token 会自动注入到 `.claude/design-tokens/` 目录，届时本文件中的规范引用会自动与 Token 体系对齐。
